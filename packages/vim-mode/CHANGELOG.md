# @dreki-gg/pi-vim-mode

## 0.1.0

### Minor Changes

- Add @dreki-gg/pi-vim-mode — an always-on vim-like modal editor extension for pi.

  - Starts in normal mode
  - Preserves pi insert-mode autocomplete and slash/file completion
  - Supports `h`, `j`, `k`, `l`, `w`, `b`, `0`, `$`, `x`, `o`, `O`, `dd`, `dw`, and `db`
  - Keeps `Esc` in normal mode wired to pi's interrupt behavior
